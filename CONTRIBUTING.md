# Contributing to StarAlive

Thanks for your interest in StarAlive. This project is currently in active MVP development by Force Creative AI.

## Reporting Issues

If you find a bug or UI issue on [staralive.vercel.app](https://staralive.vercel.app), please open a GitHub Issue with:
- What you expected to happen
- What actually happened
- Browser and device you were using
- Screenshot if possible

## Feature Requests

Open an Issue with the label `enhancement`. Describe the feature and why it would improve the artist journey.

## Development Setup

```bash
git clone https://github.com/forcecreativeai-rgb/staralive.git
cd staralive
npm install
cp .env.example .env
# Add your OpenAI API key to .env
npm run dev
```

## Branch Naming

- `feature/your-feature-name`
- `fix/bug-description`
- `chore/maintenance-task`
